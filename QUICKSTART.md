# Быстрый старт

## Шаг 1: Установка Flutter

Если Flutter еще не установлен:
1. Скачайте Flutter с https://flutter.dev
2. Распакуйте и добавьте в PATH
3. Запустите `flutter doctor` для проверки

## Шаг 2: Подготовка проекта

```bash
# Установите зависимости для мобильного приложения
cd mobile_app
flutter pub get

# Установите зависимости для десктопного приложения
cd ../desktop_app
flutter pub get
```

## Шаг 3: Запуск десктопного приложения

```bash
cd desktop_app
flutter run -d windows  # или macos, linux
```

**Важно**: Запишите IP адрес, который покажет приложение (например, 192.168.1.100)

## Шаг 4: Запуск мобильного приложения

На другом терминале или после остановки десктопного приложения:

```bash
cd mobile_app
flutter run
```

## Шаг 5: Подключение

1. В мобильном приложении введите IP адрес из шага 3
2. Порт по умолчанию: 8080
3. Нажмите "Подключиться"
4. После подключения нажмите "Начать трансляцию"
5. Разрешите доступ к камере

## Устранение проблем

### Не могу подключиться

- Убедитесь, что оба устройства в одной Wi-Fi сети
- Проверьте файрвол на ноутбуке (разрешите порт 8080)
- Попробуйте другой порт (измените в коде)

### Камера не работает

- Проверьте разрешения в настройках устройства
- Убедитесь, что камера не используется другим приложением

### Медленная трансляция

- Уменьшите качество в коде (параметр `quality` в `_processImage`)
- Уменьшите максимальный FPS (параметр `_maxFps`)
